# [CF450B](https://mirror.codeforces.com/problemset/problem/450/B)

## Tag
- `math`

## Solution
发现数列以 6 为周期循环，分类讨论即可。

## Code ($\mathcal{O}(1)$)
```cpp
void solve() {
    constexpr i64 mod = 1000000007;
    i64 x = 0, y = 0, n = 0;
    std::cin >> x >> y >> n;
    n = (n - 1) % 6;
    switch (n) {
    case 0:
        std::cout << (x + mod) % mod << '\n';
        break;
    case 1:
        std::cout << (y + mod) % mod << '\n';
        break;
    case 2:
        std::cout << (y - x + 2 * mod) % mod << '\n';
        break;
    case 3:
        std::cout << (-x + mod) % mod << '\n';
        break;
    case 4:
        std::cout << (-y + mod) % mod << '\n';
        break;
    case 5:
        std::cout << (x - y + 2 * mod) % mod << '\n';
        break;
    default:
        std::cout << 114514 << '\n';
        break;
    }
}
```
