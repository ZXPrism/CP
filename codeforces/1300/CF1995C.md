# [CF1995C](https://mirror.codeforces.com/problemset/problem/1995/C)

## Tag
- `implementation`

## Solution
按题意分类讨论模拟即可，试图去想所谓 “优雅” 的解不如暴力列出所有情况来得快。

## Code ($\mathcal{O}(n)$)
```cpp
void solve() {
    int n = 0;
    i64 k = 0;
    std::cin >> n >> k;
    std::vector<int> v(n);
    for (int i = 0; i < n; i++) {
        std::cin >> v[i];
    }
    int L = 0, R = n - 1;
    while (L <= R && k > 0) {
        if (L == R) {
            if (k >= v[L]) {
                ++L;
            } else {
                k = 0;
            }
        } else {
            if (v[L] == v[R]) {
                if (k >= 2 * v[L]) {
                    k -= 2 * v[L];
                    ++L;
                    --R;
                } else {
                    if (k == 2 * v[L] - 1) {
                        ++L;
                    }
                    k = 0;
                }
            } else if (v[L] < v[R]) {
                if (k >= 2 * v[L]) {
                    k -= 2 * v[L];
                    v[R] -= v[L];
                    ++L;
                } else {
                    if (k == 2 * v[L] - 1) {
                        ++L;
                    }
                    k = 0;
                }

            } else {
                if (k >= 2 * v[R]) {
                    k -= 2 * v[R];
                    v[L] -= v[R];
                    --R;
                } else {
                    k = 0;
                }
            }
        }
    }
    std::cout << n - (R - L + 1) << '\n';
}
```
